# 🌐 CCNA Practical Labs & Network Engineering Portfolio

> A collection of hands-on Cisco CCNA labs demonstrating practical networking skills, device configuration, routing, switching, troubleshooting, and network design using Cisco Packet Tracer.

## 📌 About This Repository

This repository documents my practical journey through the Cisco Certified Network Associate (CCNA) curriculum.

Every lab is built from scratch and includes complete configurations, verification commands, troubleshooting steps, and detailed documentation to demonstrate real-world networking concepts.

The objective is not only to prepare for the CCNA certification but also to build practical skills expected from Network Support Engineers, NOC Engineers, and IT Infrastructure professionals.
